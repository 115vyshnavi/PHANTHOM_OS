Camera Access Failed

Browsers only allow camera on HTTPS or localhost. You're opening from a local file.

Quick Fix

Open Terminal in your project folder, run:

cd Desktop/phantom && python -m http.server 8000

Then open: http://localhost:8000

Better: Deploy online

Go to app.netlify.com/drop — drag your folder — instant HTTPS link.
